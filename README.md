OBJEKTINIO PROGRAMAVIMO PAPILDOMA UZD

Tikslai:
1. Panaudoti asociatyvius konteinerius.
2. Nuskaityti faila
3. Parodyti zodzio pasikartojimo skaitciu ir jo vieta
4. Atskirti URLs

Veikimo principas:
1. Suvedamas norimas atrinkti tekstas.
2. Jis nuskaitomas apdorojimas ir isvedamas rezultatas
3. Rezultatas rodo zodi, pasikartojimo skaiciu, jo eilutes vietoje.

Naudojimo instrukcija:
1. Parsisiusti VISUS duotus failus.
2. Reikia idiegti c++ kompiliatoriu (MSYS2 MINGW64)
3. Atidaryti CMD (comand prompt)
4. Atidaryti įdiegto projekto vieta per CMD
5. Parasyti "make"
6. Paleisti atidarant exe faila arba galima parasyti komanda "make run"
7. Isvalyti sukurtus duomenis parasyti "make clean"
